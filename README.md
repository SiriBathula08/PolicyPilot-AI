# 🚀 PolicyPilot AI
### AI-Powered Government Scheme Recommendation System using Retrieval-Augmented Generation (RAG)

PolicyPilot AI is an AI-powered recommendation system that helps users discover Indian government schemes based on their eligibility.

The application combines **Retrieval-Augmented Generation (RAG)** with **semantic search** to retrieve relevant policy documents and provide personalized scheme recommendations. Instead of relying on keyword matching, it uses vector embeddings and similarity search to improve retrieval quality.

## ✨ Features

- 🤖 AI-powered government scheme recommendations
- 🔍 Semantic search using vector embeddings
- 📚 Retrieval-Augmented Generation (RAG)
- 🎯 Eligibility-based recommendation engine
- ⚡ Fast similarity search using FAISS
- 💬 Interactive Streamlit web application

---

## 🏗️ System Workflow

1. User enters eligibility details or a policy-related query.
2. The query is converted into vector embeddings.
3. FAISS retrieves the most relevant policy documents.
4. The RAG pipeline generates personalized recommendations.
5. The results are displayed through the Streamlit interface.
